# Student Performance Prediction 📚

A Machine Learning project that predicts **student academic performance** based on various demographic, social, and academic factors using predictive modeling techniques.

## 📌 Project Overview
This project analyzes student-related data and predicts academic performance using Machine Learning algorithms. It helps understand how different factors like study time, attendance, parental education, and other features can impact student outcomes.

## 🚀 Features
- Student performance prediction
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Data visualization
- Model training and evaluation
- Performance analysis using ML techniques

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Machine Learning

## 📂 Dataset
The dataset contains student-related information such as:
- Study time
- Attendance
- Previous scores
- Parental education
- Age
- Gender
- Other academic/demographic factors

## ⚙️ Workflow
1. Import dataset
2. Data preprocessing & cleaning
3. Exploratory Data Analysis (EDA)
4. Feature selection
5. Train-test split
6. Model training
7. Prediction & evaluation

## 🤖 Machine Learning Model
This project uses Machine Learning algorithms to predict student performance based on input features.

Possible models:
- Linear Regression
- Decision Tree
- Random Forest
- Logistic Regression (if classification)

## 📊 Results
The model predicts student performance with good accuracy and helps identify important factors affecting academic outcomes.

## ▶️ Installation
Clone the repository:

```bash
git clone https://github.com/praaachii18/Student_Performance_Prediction.git
```

```bash
pip install pandas numpy matplotlib scikit-learn
```

```bash

```
